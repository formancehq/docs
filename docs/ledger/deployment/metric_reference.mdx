# Ledger Metric References

This page provides a reference for the metrics that are published by the Ledger.

| Name | Type | Unit | Description |
|------|------|------|-------------|
| `ledger.api.time` | Histogram | `ms` | Latency of API requests. |
| `ledger.api.status` | `int64` counter | `N/A` | Count of API requests by status code. |
| `ledger.api.ledgers` | `int64` counter | `N/A` | Number of active Ledger |
