## Prerequisites

Before you begin, you need to have a Atlar account. Make sure you have a Atlar API key and secret, with the least amount of permissions required to access the functionality of the Atlar Connector you plan to use.

## Installation

<StackCmd>

```shell
fctl payments connectors install atlar config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/atlar" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```
