## Prerequisites

Before you begin, you need to have a Currencycloud account. Make sure you have a Currencycloud API key and secret, with the least amount of permissions required to access the functionality of the Currencycloud Connector you plan to use.

## Installation

<StackCmd>

```shell
fctl payments connectors install currencycloud config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/currencycloud" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```
