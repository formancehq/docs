## Prerequisites

Before you begin, you need to have a Wise account. Make sure you have a Wise API key and secret, with the least amount of permissions required to access the functionality of the Wise Connector you plan to use.

<ApiWarning/>

<Wise/>

## Installation

<StackCmd>

```shell
fctl payments connectors install wise config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/wise" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```

</StackCmd>

With config.json containing:

<ReactJson
  src={WiseConfig}
  theme="ocean"
  displayDataTypes={false}
  collapsed={1}
  name={false}
  indentWidth={2}
/>
