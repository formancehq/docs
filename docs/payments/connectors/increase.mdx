## Prerequisites

Before you begin, you need to have a Increase account. Make sure you have a Increase API key and secret, with the least amount of permissions required to access the functionality of the Increase Connector you plan to use.

## Installation

<StackCmd>

```shell
fctl payments connectors install increase config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/increase" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```
