## Prerequisites

Before you begin, you need to have a Moneycorp account. Make sure you have a Moneycorp API key and secret, with the least amount of permissions required to access the functionality of the Moneycorp Connector you plan to use.

## Install

<StackCmd>

```shell
fctl payments connectors install moneycorp config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/moneycorp" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```

</StackCmd>

With config.json containing:

```json
{"apiKey":"string","clientID":"string","endpoint":"string","name":"string","pageSize":"25","pollingPeriod":"2m"}
```
