## Prerequisites

Before you begin, you need to have a Bankingcircle account. Make sure you have a Bankingcircle API key and secret, with the least amount of permissions required to access the functionality of the Bankingcircle Connector you plan to use.

<ApiWarning/>

## Setup

<Bankingcircle/>

<StackCmd>

```shell
fctl payments connectors install bankingcircle config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/bankingcircle" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```

</StackCmd>

With config.json containing:

```json
{"authorizationEndpoint":"string","endpoint":"string","name":"string","pageSize":"25","password":"string","pollingPeriod":"2m","userCertificate":"string","userCertificateKey":"string","username":"string"}
```
