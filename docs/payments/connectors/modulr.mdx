## Prerequisites

Before you begin, you need to have a Modulr account. Make sure you have a Modulr API key and secret, with the least amount of permissions required to access the functionality of the Modulr Connector you plan to use.

## Install

<StackCmd>

```shell
fctl payments connectors install modulr config.json
```

```shell
curl -X POST "https://$STACK.formance.cloud/api/payments/connectors/modulr" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer $TOKEN" \
-d @config.json
```

</StackCmd>

With config.json containing:

```json
{"apiKey":"string","apiSecret":"string","endpoint":"string","name":"string","pageSize":"25","pollingPeriod":"2m"}
```
