# Getting help

Are you stuck? Have a technical question not well answered in these docs? Run into a problem you can't solve? We can help!

Come chat with us! We have [a Discord server](https://discord.com/invite/xyHvcbzk4w) where we and others using Numary Ledger hang out.

Or, if you prefer, you can also [Tweet at us](https://twitter.com/numaryhq).
